Strategic Career Plan

Year 1–2: The Launch Pad

Path: Enroll in a Computer Science BS program at Arizona State University to build a deep foundation in algorithms and systems.

Action Items: Join the local Game Dev club, complete the Google UX Design Certificate during summer break to bridge the gap between engineering and design.

Year 3–5: The Pivot

Internships: Target a Junior Software Engineer internship at a local tech hub (e.g., Intel or GoDaddy in AZ) to gain real-world Agile experience.

Standing Out: Build a specialized GitHub portfolio featuring one full-stack web application and one playable game prototype built in Unreal Engine.

Advanced Credential: Obtain the AWS Certified Developer – Associate to demonstrate cloud-native development mastery.

The Dream Job: Senior Game Systems Designer

Environment: Hybrid – 3 days in a creative studio for collaborative "whiteboarding," 2 days remote for deep focus on game balance spreadsheets and logic.

Daily Responsibilities: Designing combat math, scripting character abilities in C++, and analyzing player playtest data to "find the fun."

Technologies: Unreal Engine 5, C++, Jira for project tracking, and Python for data analysis tools.

Impact: Creating immersive digital worlds that provide players with a sense of agency and emotional connection, while mentoring junior designers to foster a more inclusive gaming industry.
